Um plugin para adicionar um editor de código no front-end usando um shortcode.

== Descrição ==
Este plugin permite aos usuários editar e salvar códigos HTML, CSS e JavaScript diretamente no front-end de um site WordPress. Utiliza o editor Ace para uma experiência de edição avançada.

== Instalação ==
1. Faça o upload da pasta `seu-plugin-editor-codigo` para o diretório `/wp-content/plugins/`.
2. Ative o plugin através do menu 'Plugins' no WordPress.
